## How to Run Flask Projects

### 1. Clone the repository

```
git clone https://github.com/c-angel11/Python-Internship-Tasks.git
```

### 2. Navigate to the required task folder

Example:

```
cd Task3
```

### 3. Install Flask

```
pip install flask
```

### 4. Run the application

For Task 3:

```
python app1.py
```

### 5. Open in browser

```
http://127.0.0.1:5000
```
